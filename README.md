# 📊 Placement Preparation Tracker

A full-stack web application built using Flask and MySQL to help students track coding problems and company applications during placement preparation.

---

## 🚀 Live Demo
https://placement-preparation-tracker-5dud.onrender.com

---

## ✨ Features

### 🔐 User Authentication
- Register new users
- Login / Logout
- Session management

### 🧠 Problem Tracker
- Add coding problems
- Edit / Delete problems
- Track difficulty, platform, and status
- Filters and sorting
- Progress dashboard

### 🏢 Company Tracker
- Add company applications
- Track status (Applied, OA, Interview, Selected, Rejected)
- Edit / Delete entries
- Search and filter

### 📊 Dashboard
- Total problems solved
- Progress percentage
- Status-based analytics

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: MySQL
- Deployment: Render
